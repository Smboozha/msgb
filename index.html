Как запустить и протестировать
  1. Откройте эту страницу в двух вкладках браузера (или на двух устройствах).
  2. В каждой вкладке вверху отобразится уникальный ID (Peer ID). Скопируйте ID из одной вкладки.
  3. В другой вкладке вставьте скопированный ID в поле «ID собеседника» и нажмите «Подключиться».
  4. После подключения можно отправлять сообщения — они будут доставляться напрямую через P2P.
  5. Если страница размещена на GitHub Pages, откройте её в двух окнах браузера и повторите шаги 2–4.

Примечание: Для работы необходим доступ к серверам PeerJS (по умолчанию 0.peerjs.com). Если он недоступен, P2P-соединение может не установиться.
